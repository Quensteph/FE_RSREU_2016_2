Initial value is 0

  • Type calculator.add(x) to perform an addition
  • Type calculator.subtract(x) to perform a subtraction
  • Type calculator.multiply(x) to perform a multiplication
  • Type calculator.divide(x) to perform a division

calculator supports multiple inputs. Example syntax: calculator.add(x)(y)

To display current resulting value, type calculator.getResult()

To reset current resulting value, type calculator.reset()
